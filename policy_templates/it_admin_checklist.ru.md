# Чек-лист ИТ-администратора

## Ежедневно
- [ ] Проверка доступности интернет-канала.
- [ ] Проверка логов на наличие аномалий (DDoS, попытки взлома).

## Еженедельно
- [ ] Обновление антивирусных баз.
- [ ] Резервное копирование конфигураций роутеров.

## Ежемесячно
- [ ] Проверка обновлений прошивок (OpenWrt/RouterOS).
- [ ] Смена паролей Wi-Fi (если требуется).
- [ ] Тестовое восстановление из бэкапа.
