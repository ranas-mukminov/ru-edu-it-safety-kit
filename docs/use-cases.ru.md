# Сценарии использования

## 1. Малая школа (Single ISP)
- **Описание**: Одно здание, один интернет-провайдер.
- **Задача**: Разделить трафик учителей и учеников, заблокировать нежелательный контент.
- **Решение**:
    - 1 роутер (MikroTik/OpenWrt/pfSense).
    - 3 VLAN: Admin (управление), Teachers (журналы), Students (фильтруемый доступ).
    - DNS-фильтрация (SkyDNS/Yandex) принудительно для VLAN Students.

## 2. Колледж с филиалами (Multi-building)
- **Описание**: Главный корпус + общежитие или удаленный филиал.
- **Задача**: Единая сеть, контроль доступа из общежития.
- **Решение**:
    - VPN-туннель (IPsec/WireGuard) между зданиями.
    - Единая политика фильтрации.
    - Логирование доступа в интернет (для расследования инцидентов).
