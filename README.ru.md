# ru-edu-it-safety-kit

**Базовый набор для ИТ-безопасности российских школ и колледжей.**

Этот проект предоставляет набор инструментов и конфигураций для создания базовой, безопасной и соответствующей требованиям ИТ-инфраструктуры в образовательных учреждениях.

> **Проект поддерживается run-as-daemon.ru**. Для продакшен-настроек, аудитов и сопровождения ИТ-безопасности школ и колледжей переходите на [https://run-as-daemon.ru](https://run-as-daemon.ru).

## Обзор

**Для кого:** ИТ-администраторы и сотрудники школ и колледжей.

**Основные возможности:**
- **Моделирование профиля школы:** Описание топологии сети (провайдеры, здания, VLAN) в простом YAML формате.
- **Планировщик мер безопасности:** Автоматическая генерация чек-листа технических и организационных мер.
- **Генератор памяток:** Создание готовых к печати инструкций для родителей, учеников и учителей (на русском языке).
- **Шаблоны конфигураций:** Готовые примеры настроек для OpenWrt, pfSense и RouterOS.
- **Стек логирования:** Примеры конфигурации для централизованного сбора логов (Loki/ELK).

## Архитектура

Набор состоит из:
1.  **Python CLI (`ru-safety`)**: Основной инструмент для обработки профилей и генерации материалов.
2.  **YAML Профили**: Декларативное описание инфраструктуры школы.
3.  **Шаблоны**: Шаблоны конфигураций и документов.

## Установка

### Требования
- Python 3.9+

### Локальный запуск

1.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/run-as-daemon/ru-edu-it-safety-kit.git
    cd ru-edu-it-safety-kit
    ```

2.  Создайте виртуальное окружение и установите зависимости:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -e .[dev]
    ```

## Использование

### 1. Проверка профиля школы
```bash
ru-safety validate configs/topologies/single_isp_school.yml
```

### 2. Генерация плана безопасности
```bash
ru-safety plan configs/topologies/single_isp_school.yml
```

### 3. Генерация памяток
```bash
ru-safety generate-leaflets configs/topologies/single_isp_school.yml --output-dir ./dist
```

## Тестирование и качество

Запуск всех тестов:
```bash
pytest
```

Запуск линтеров и проверок безопасности:
```bash
ruff check .
black --check .
bandit -r src/
```
